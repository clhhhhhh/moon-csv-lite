项目简介

moon-csv-lite 是一个以 MoonBit 为主要实现语言的 CSV 数据质量工程工具包。它不只提供 CSV 解析和导出，还围绕“CSV 作为可发布数据资产”提供 schema 推断、质量评分、质量问题检测、数据漂移报告、质量门禁、质量契约、数据护照、图表预览、命令行工具和浏览器 Playground。适用场景包括配置表、产品目录、实验/测量数据、日志导出、游戏数据表、教学数据集和 CI 中需要验收的 CSV 文件。

核心功能范围

  • CSV parse/stringify，支持空字段、引号字段、转义引号、引号内逗号/换行、LF/CRLF、TSV、分号和竖线分隔格式。
  • 自动 dialect sniffing、checked parse、行宽错误和未闭合引号等可恢复解析问题报告。
  • header-aware table、Markdown/JSON Lines/HTML 导出、列选择、过滤、排序、清洗、去重、join、union、transpose 和 group-by 聚合。
  • schema inference、required/type/range/allowed-values/unique-key 校验、列画像、缺失值统计和表级质量问题检测。
  • 0-100 数据质量评分、Markdown/HTML/JSON audit report 和面向修复的 recommendations。
  • CSV Drift Report 与 CSV Quality Gate，用于 CI 中比较 baseline/candidate 并对删除列、类型变化、缺失值增加和质量分下降给出 pass/fail。
  • CSV Quality Contract，可从样本推断契约，也可用 CSV 规则表声明 min_score、required、type、min/max、allowed、unique 等质量规则。
  • CSV Data Passport，将数据集名称、dialect、行列规模、稳定形状指纹、质量分、schema 摘要、列元数据和建议打包为 Markdown/JSON 交付物。
  • cmd/csvlite CLI、文件审计脚本、fixture 测试集、可重新生成示例报告和 MoonBit JS backend 驱动的中文浏览器 Playground。

参考说明

本项目不是移植项目，没有直接复制或改写某个已有仓库。MoonBit 生态已有 NyaCSV、csv_parser、MoonFrame、mooncharts 等解析、表格和图表项目；moon-csv-lite 的独立贡献是围绕 CSV 数据资产形成完整质量工程工作流：审计报告、数据护照、质量契约、漂移检测、质量门禁、CLI、fixtures、CI、示例文档和浏览器预览。
